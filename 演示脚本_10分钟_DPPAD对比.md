### 【1:00-3:00】环节2: DPPAD算法原理 (2分钟)

**操作**: 打开 PPT 或代码展示

**讲解词**:
> "DPPAD-IE 算法包含两个核心阶段：
> 
> 1. **置乱阶段**: 使用混沌映射生成置乱序列，打乱像素位置
> 2. **扩散阶段**: 采用双向扩散机制，实现雪崩效应
> 
> 算法特点：
> - 基于 2D-CGHM 混沌映射生成密钥流
> - 阿诺德映射实现双向扩散
> - 高度的密钥敏感性和明文敏感性"

**屏幕展示**:
```
原始图像 → [置乱] → 置乱图像 → [扩散] → 加密图像
                ↑                    ↑
            混沌序列            密钥流 K1,K2,K3
```

**关键代码片段** (可选展示):
```python
# 置乱阶段
perm_indices = get_permutation_indices(L, x1, r1)
P_permuted = P_flat[perm_indices]

# 扩散阶段
K1, K2, K3 = generate_2d_cghm(x3, y3, a1, b1, L)
C_flat = arnold_diffusion_forward(P_permuted, K1, K2, K3, a, b)
```
