# 图像加密算法测评系统 - 10分钟答辩指南

## 时间分配 (总计10分钟)
- **系统介绍** (2分钟)
- **核心功能演示** (4分钟)
- **技术亮点** (2分钟)
- **结果展示** (2分钟)

---

## 1. 系统介绍 (2分钟)

### 开场白
"各位老师好，我今天展示的是**图像加密算法测评系统**，这是一个专门用于评估和比较不同图像加密算法性能的综合平台。"

### 系统概述
- **目标**: 为图像加密算法提供标准化的测评环境
- **特点**: 
  - 支持多种加密算法
  - 自动化性能评估
  - 直观的图形界面
  - 详细的安全性分析

### 技术栈
- **界面**: PySide6 (现代化GUI)
- **算法**: Python + Numba加速
- **评估**: 熵值、相关性、NPCR/UACI等指标

---

## 2. 核心功能演示 (4分钟)

### 演示流程
1. **启动系统** (30秒)
   ```bash
   python main_pyside6.py
   ```
   - 展示系统启动界面
   - 显示已加载的6个算法

2. **算法选择** (30秒)
   - 展示算法下拉菜单
   - 介绍可用算法：
     - Fast Chaos Encryptor (混沌加密)
     - Numba Accelerated Encryptor (高性能)
     - Simple XOR (基础对比)

3. **图像加载与加密** (1.5分钟)
   - 加载测试图像 (brain1.jpg 或其他)
   - 选择 Fast Chaos Encryptor
   - 点击加密，展示：
     - 原图 → 加密图 → 解密图
     - 实时性能数据

4. **评估结果查看** (1.5分钟)
   - 展示评估报告
   - 重点指标：
     - **熵值**: 原图 vs 密文 (接近8.0为佳)
     - **相关性**: 原图 vs 密文 (接近0为佳)
     - **NPCR**: >99% (抗差分攻击)
     - **UACI**: 33%左右 (均匀性)
     - **加密速度**: 像素/秒

## 3. 技术亮点 (2分钟)

### 架构设计
- **插件化架构**: 算法可插拔，易于扩展
- **多线程处理**: 界面响应 + 后台计算分离
- **模块化设计**: 核心、界面、算法分离

### 性能优化
- **Numba JIT编译**: 接近C语言性能
- **向量化操作**: NumPy优化的矩阵运算
- **内存管理**: 大图像的高效处理

### 算法特色
- **混沌加密**: 基于混沌映射的高安全性
- **RGB通道处理**: 支持彩色图像完整加密
- **可逆性验证**: 确保完美解密恢复

## 4. 结果展示 (2分钟)

### 性能对比
```
算法名称                加密速度        安全指标
Fast Chaos Encryptor   高性能         高安全性
Numba Accelerated      超高性能       高安全性  
Simple XOR             极高性能       基础安全性
```

### 安全性评估
- **熵值分析**: 密文熵接近理论最大值8.0
- **相关性分析**: 相邻像素相关性接近0
- **差分分析**: NPCR>99%, UACI≈33%
- **完美恢复**: MSE=0，无信息损失

### 实际应用价值
- **算法研究**: 为新算法提供标准测评
- **性能比较**: 客观评估不同算法优劣
- **教学工具**: 直观展示加密原理和效果
